# nfr.md

## 📌 Purpose of this File
Defines **Non-Functional Requirements (NFRs)**.

👉 Why this matters:
- Drives architecture choices
- Influences deployment, scaling, security

---

## ⚙️ NFRs

| NFR | Description | Business Impact |
|-----|------------|----------------|
| Performance | Real-time decision (< 2 sec) | Better customer experience |
| Scalability | Handle high volume | Business growth |
| Availability | 99.9% uptime | Reliability |
| Security | Data protection | Compliance |
| Explainability | AI decisions must be explainable | Regulatory requirement |
